Perangkat Pemantauan Kualitas Air Irigasi Terintegrasi dengan IoT ğŸ’§

Full Source Code (Kontak Kami) Rp.4.999k* Full Embedded System

Alat ini akan memonitor kadar keasaman (pH), TDS (Total Disolved Solid), serta suhu air irigasi yang ingin diuji, data hasil uji akan ditampilkan dalam aplikasi android melalui koneksi bluetooth serta dapat dimonitoring secara real-time melalui website yang tersedia.

  • Menggunakan ESP32 (C++) sebagai kontroler dan konektivitas WiFi dan Bluetooth
  • Nilai sensor akan diperbarui sesuai dengan waktu yang diatur melalui aplikasi android
  • Desain alat kompak dan portable sehingga mudah untuk dioperasikan
  • Data dapat disimpan dalam bentuk tabel pada excel untuk memudahkan pengolahan data

Daya disuplai melalui baterai LiPo dengan data 720 mAh. Terdapat video demo alat serta untuk informasi lebih lanjut dapat langsung menghubungi kami ğŸ“§.

Sistem Otomatis Pencampuran Nutrisi Hidroponik ğŸŒ±

Full Source Code (Kontak Kami) Rp.1.799k* Full Embedded System

Sistem pada alat ini akan secara otomatis bekerja sesuai waktu yang telah ditentukan sebelumnya, berdasar pada keasaman (pH), TDS (Total Disolved Solid), serta ppm nutrisi yang telah diatur.

  • Menggunakan STM32 (C++) sebagai kontrol sistem
  • Bekerja sesuai waktu yang ditentukan tiap harinya
  • Nilai sensor ditampilkan melalui LCD serta aliran nutrisi dikontrol dengan selenoid

Perangkat ini diimplemantasikan dalam sistem yang besar tergantung dari area hidroponik yang dipakai, penambahan beberapa sistem antara lain IoT maupun konektivitas lainnya dimungkinkan guna akses perangkat yang lebih luas ğŸŒ.
